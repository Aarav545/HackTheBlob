How it works?

Logistic Regression

Logistic Regression also know as the sigmoid function is used commonly. Don't be fooled by the name this is not regression but classification. From keras deep learning to many machine learning tasks. The formula being depicted here multiples the X(input) by the corressponding coefficents and is raised to the negative exponent of 1.


Why is it only between 0 and 1?

Continous output will be an impossible achievement because of the lack of data. Using a bound of 50 to decide between 0 and 1.(Whether the day is good or bad)

Why not use a deep learning model?

Machine learning models can be milisecond predictions unlike deep learning models. Deep learning models need a lot of data as they are very complex.
